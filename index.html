Swords of Exile

A remake of the indie RPG 'Blades of Exile', originally released in 1997. It was written from the ground up in C#/.NET, using MonoGame as the graphics engine. The original game featured a scenario editor, allowing users to write their own custom adventures. This version features Python scripting within .NET using IronPython, to replicate the customised behaviour in the original scenarios.

Pose Estimator

For my final year Creative Computing project I created a pose estimation library for Processing using a freely available TensorFlow model, and used it to create a hands-free digital musical instrument, the Pose Harp.

Maze Run

While no longer at the cutting-edge, attempting to squeeze enough power out of the ZX Spectrum's Z80 processor to run an action game is no easy feat. Maze Run was written entirely in Z80 Assembler.

Op Art Virtual Gallery

For one of my Creative Computing projects I created a 3D virtual art gallery based on the abstract painting of artist Bridget Riley. A variety of transformations within the gallery blur the boundaries between the gallery pieces and the geometry of the gallery itself.
